Announces Kong Agent Gateway as a capability of Kong AI Gateway for governing
agent-to-agent (A2A) communication. Covers agent identity, real-time traffic
inspection for prompt-injection, unified observability across LLM/MCP/A2A
flows, and per-agent cost allocation.
